# Uiverse.io Components Practice

This project is a collection of tested and customized UI components from [Uiverse.io](https://uiverse.io/elements).

## Overview
The goal of this practice was to explore, test, and integrate different modern CSS components from Uiverse.io. The project includes buttons, switches, cards, and other UI elements combined into a single responsive layout.

## Features
- Multiple UI components with unique styles
- Responsive flexbox layout
- Proper spacing between components
- Reusable and clean CSS
- Easy customization for future projects

## Files
- index.html – contains all imported components and structure
- style.css – contains layout, spacing, and component styles

## How to Run
1. Clone or download the project.
2. Open the folder in VS Code or any code editor.
3. Open **index.html** in a browser.

## Credits
All components are sourced and adapted from [Uiverse.io](https://uiverse.io/elements), an open-source platform for ready-made CSS and Tailwind UI elements.
